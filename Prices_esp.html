¿Qué software se necesita para usar Smart TIO?

Smart TIO funciona sin problemas en Windows 10 y superior. No podemos garantizar que Smart TIO funcione en versiones inferiores de Windows.
Si tienes alguna duda tras , siempre puedes con nuestro equipo.

La versión para Mac estará dispoible muy pronto.

¿Está disponible Smart TIO en mi país?

Hemos creado un listado de paises en los que Smart TIO está actualemente disponible. Puedes descargarlo Este PDF también contiene información sobre los datos que estamos utilizando para cada módulo.
Si no encuentra su país, póngase en contacto con nosotros y comprobaremos la disponibilidad de datos para su país. Después, le informaremos sobre los posibles ajustes necesarios y el precio de Smart TIO para su país. Continuamos trabajando para llegar cada día más lejos.

¿Durante cuánto tiempo puedo usar Smart TIO una vez lo compre?

Actualmente tenemos licencias para el uso de máximo dos años pero si lo necesitas podremos ampliar tu licencia o incluso hacerla permanente.

Nuestro equipo le enviará un código de activación así como un enlace para descargar el paquete de instalación. Con este código podrá utilizar Smart TIO para todos los usuarios para los que haya adquirido la aplicación.

¿Necesito internet para poder trabajar con Smart TIO?

Aunque no es necesario tener conexión a internet para utilizar Smart TIO, si lo es para descargarlo e instalarlo.

¿Puede AYeconomics tener acceso a los datos de mi simulación?

AYeconomics no puede acceder a los datos ya sean de la versión demo o de las versiones completas de Smart TIO.

¿Se realizan acualizaciones en Smart TIO?

Se realizan actualizaciones cada vez que se encuentren errores o se realicen mejoras así como cada vez que ls datos sean actualizados. Se os enviará información de como llevar a cabo las actualizaciones así como de los cambios y mejoras realizadas.

¿Cada cuanto tiempo se realiza una actualización?

Está previsto una actulización anual que incluya nuevas funcioanlidades, cálculos y mejoras. El equipo enviará infomación sobre las ventajas de la nueva versión y los precios para las nuevas verisones si esto fuera preciso.
